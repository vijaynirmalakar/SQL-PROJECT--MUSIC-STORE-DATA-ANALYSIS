🎵 Music Store Data Analysis — SQL Project
A complete end-to-end SQL data analysis project on a Music Store database. The project involves importing, querying, and deriving business insights from a relational database using MySQL

🗄️ Database Schema
The database contains 11 tables covering 
artist, 
album, 
track, 
genre,
media_type,
customer, 
employee
invoice,
invoice_line
playlist,
playlist_track
